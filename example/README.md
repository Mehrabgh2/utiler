# utiler examples

This repo includes **copy/paste snippets** demonstrating usage of the utilities in `lib/src/`.

There is no runnable Flutter app scaffold included by default. Import the snippet widgets into your own Flutter app.

## Categories

- `example/async/` (Debouncer, Throttler, Retry)
- `example/concurrency/` (BatchExecutor, ParallelExecutor)
- `example/core/` (Either, Guard, InternetConnectivity, LazyValue, LifecycleHandler)
- `example/database/` (Database, JsonDatabaseData, SecureDatabaseData)
- `example/extension/` (String/Num/List/Map extensions + ColorExtension + gaps)
- `example/logger/` (Logger, PrettyLogger, StopwatchLogger, LoggerConsole)
- `example/service/` (ApiService)
- `example/ui/` (widgets + UI helpers)
